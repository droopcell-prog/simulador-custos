📱 Gerar APK para Android

📋 Pré-requisitos

Você já tem os arquivos necessários na pasta sistema/:

  • ✅ simulador-custos.html — aplicativo completo
  • ✅ manifest.json — manifesto PWA
  • ✅ sw.js — service worker (offline)
  • ✅ icons/ — ícones em vários tamanhos

Escolha um dos métodos abaixo para gerar o APK.

Fácil Método 1: PWABuilder (recomendado)

Ferramenta gratuita da Microsoft que converte PWA em APK. Não precisa instalar nada.

1

Hospede temporariamente os arquivos em um servidor local ou gratuito.

Opções: Netlify Drop (https://app.netlify.com/drop), GitHub Pages, ou Python server (python -m http.server 8080).

2

Acesse https://pwabuilder.com

3

Cole a URL do seu app hospedado e clique em "Start".

4

Clique em "Package for Stores" → depois em "Android".

O PWABuilder gera o APK assinado automaticamente.

5

Baixe o APK e instale no seu Android.

Talvez precise ativar "Instalar de fontes desconhecidas" nas configurações.

🔗 Abrir PWABuilder

Fácil Método 2: WAPK Generator (online)

Alternativa gratuita que não precisa de hospedagem — você envia o HTML diretamente.

1

Acesse https://appmaker.xyz/pwa-to-apk

2

Informe a URL do seu app e clique em "Generate APK".

3

Baixe o APK gerado e instale no celular.

Avançado Método 3: Android Studio (controle total)

Para quem quer personalizar ícone, splash screen, permissões, etc.

1

Instale o Android Studio em https://developer.android.com/studio

2

Crie um novo projeto "Empty Activity".

3

Adicione um WebView que carrega o arquivo local:

webView.loadUrl("file:///android_asset/simulador-custos.html")

4

Copie simulador-custos.html + icons/ para src/main/assets/.

5

Ative o JavaScript e o armazenamento interno no WebView:

webView.settings.javaScriptEnabled = true
webView.settings.domStorageEnabled = true

6

Gere o APK assinado em Build → Generate Signed Bundle / APK.

💡 Dica: PWA já instala sem APK

O próprio navegador Chrome no Android já permite instalar o app na tela inicial sem precisar de APK. Basta:

  1. Abrir o simulador-custos.html no Chrome
  2. Clicar nos três pontinhos ⋮
  3. Selecionar "Adicionar à tela inicial"

O app instalado via PWA funciona offline, ocupa pouco espaço e tem o mesmo comportamento de um app nativo.

Todos os métodos são gratuitos. Nenhum servidor ou hospedagem permanente é necessária.
